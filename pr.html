GitHub Token

This uses the shared github_token localStorage key used by the other GitHub tools. The token is only sent to GitHub API requests. Without a token the page loads a read-only view of public PRs (all threads shown; resolved/hidden state and reply/resolve/hide are unavailable). With a token it uses the GraphQL API to show unresolved state and lets you reply, resolve, and hide — use a token with repo scope for those actions.
