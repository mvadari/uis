GitHub Token

This uses the shared github_token localStorage key used by the other GitHub tools. The token is only sent to GitHub API requests. A token is required because the GraphQL API (needed to read unresolved threads and reply/resolve) does not allow anonymous access. Use a token with repo scope to reply and resolve.
